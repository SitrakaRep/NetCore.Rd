Clone repository and after change your specific connectionstring database at appsettings.json file into NetCore.Rd.Web project. 

Open CMD at the same folder and run the following commands : 
 - dotnet restore
 - dotnet ef database update. 
 
Then run your project.

Link : 

 - Apartments List : ~/apartments
 - Owners List : ~/owners
